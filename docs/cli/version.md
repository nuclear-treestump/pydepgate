# pydepgate version

Print the installed pydepgate version.

```
pydepgate version
```

## Output

```
pydepgate x.y.z
```

## Alternatives

The top-level `--version` flag produces the same output:

```bash
pydepgate --version
```

Both write to stdout and exit with code 0.
